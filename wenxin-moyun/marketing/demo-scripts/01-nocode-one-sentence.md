# Demo Script #1: NoCode — One Sentence, Six Agents (30s)

> **Duration**: 30 seconds
> **Mode**: Canvas → Run
> **Audience**: Non-technical users, product managers, AI researchers
> **Key Message**: Zero code, one sentence input, full cultural evaluation pipeline

---

## Pre-Recording Checklist

- [ ] Open https://vulcaart.art in Chrome (1920×1080, 100% zoom)
- [ ] Navigate to Canvas page → ensure Run mode is active
- [ ] Clear any previous sessions (fresh state)
- [ ] Screen recorder: 1080p, 30fps minimum
- [ ] Microphone off (voice-over added in post)
- [ ] Browser bookmarks bar hidden
- [ ] No other tabs visible

---

## Shot-by-Shot Script

### [0:00–0:03] Opening — Run Mode Empty State

**Screen**: Run mode initial view — empty canvas with IntentBar at top, TopologyViewer showing 6 gray nodes (Scout → Router → Draft → Critic → Queen → Archivist)

**Action**: None (hold shot)

**Annotation**: `VULCA Canvas — Run Mode` (top-left, fade in)

**Voice-over cue**: "One sentence. Six AI agents. Real cultural evaluation."

---

### [0:03–0:08] Type the Prompt

**Screen**: IntentBar text field

**Action**: Click IntentBar → type `bamboo forest under moonlight` (natural typing speed, ~1 char/100ms)

**Annotation**: Arrow pointing to IntentBar: `Your creative intent →`

**Voice-over cue**: "Just describe what you want to create."

---

### [0:08–0:10] Select Tradition

**Screen**: Tradition dropdown (right side of IntentBar)

**Action**: Click tradition dropdown → scroll to "Chinese Xieyi" → select

**Annotation**: `Choose a cultural lens` (brief flash near dropdown)

---

### [0:10–0:12] Hit Create

**Screen**: IntentBar with filled prompt + selected tradition

**Action**: Click the "Create" button (blue, right side)

**Annotation**: Flash effect on button press

**Voice-over cue**: "Hit Create."

---

### [0:12–0:18] Agent Pipeline Execution

**Screen**: TopologyViewer — 6 nodes animate sequentially

**Action**: Watch as nodes light up one by one:
1. **Scout** (0:12) — blue pulse → green check ✓
2. **Router** (0:13) — blue pulse → green check ✓
3. **Draft** (0:14–0:15) — blue pulse, takes slightly longer → green check ✓
4. **Critic** (0:15–0:16) — blue pulse → green check ✓
5. **Queen** (0:16–0:17) — blue pulse → green check ✓
6. **Archivist** (0:17–0:18) — blue pulse → green check ✓

**Annotation** (progressive, each appears as node activates):
- `Scout: Analyzes intent`
- `Router: Selects cultural pipeline`
- `Draft: Generates candidates`
- `Critic: Evaluates L1-L5`
- `Queen: Final judgment`
- `Archivist: Stores & learns`

**Voice-over cue**: "Six specialized agents work together — analyzing, routing, creating, evaluating, judging, and learning."

---

### [0:18–0:22] Candidate Gallery Appears

**Screen**: CandidateGallery panel fills with generated image(s) + score badges

**Action**: Gallery slides in from right. Score badges animate (number counts up).

**Annotation**: `Generated candidates with cultural scores` (below gallery)

**Voice-over cue**: "Candidates generated and scored — not just aesthetically, but culturally."

---

### [0:22–0:26] Radar Chart

**Screen**: CriticRadarChart fills with L1-L5 dimensions

**Action**: Radar polygon draws itself (animated fill). Hover over a dimension to show tooltip.

**Annotation**: Label each axis as it appears:
- `L1: Recognition`
- `L2: Technique`
- `L3: Context`
- `L4: Meaning`
- `L5: Innovation`

**Voice-over cue**: "Five-level cultural evaluation — from basic recognition to deep innovation."

---

### [0:26–0:28] Round Timeline

**Screen**: RoundTimeline panel showing round cards with decisions

**Action**: Scroll briefly through timeline cards

**Annotation**: `Full decision trace — every agent's reasoning visible`

**Voice-over cue**: "Every decision is traceable."

---

### [0:28–0:30] Final Score & Close

**Screen**: Overall score summary + radar chart

**Action**: Hold on final state. Slight zoom on score.

**Annotation**:
- Center: `One sentence → Full cultural evaluation`
- Bottom: `vulcaart.art` + `Open Source`

**Voice-over cue**: "One sentence in. Full cultural evaluation out. No code required."
