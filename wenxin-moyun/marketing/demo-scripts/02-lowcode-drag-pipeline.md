# Demo Script #2: LowCode — Drag & Drop Pipeline (30s)

> **Duration**: 30 seconds
> **Mode**: Canvas → Edit → Run
> **Audience**: Developers, AI engineers, tech-savvy product managers
> **Key Message**: Visual pipeline builder — customize your evaluation flow without writing code

---

## Pre-Recording Checklist

- [ ] Open https://vulcaart.art in Chrome (1920×1080, 100% zoom)
- [ ] Navigate to Canvas page → switch to Edit mode
- [ ] Clear canvas (no existing nodes)
- [ ] Screen recorder: 1080p, 30fps minimum
- [ ] Ensure smooth mouse movements (consider mouse smoothing software)
- [ ] Browser bookmarks bar hidden

---

## Shot-by-Shot Script

### [0:00–0:03] Opening — Edit Mode Canvas

**Screen**: Edit mode — React Flow canvas with grid background, CanvasToolbar on left, empty workspace

**Action**: None (hold shot, subtle mouse movement)

**Annotation**: `VULCA Canvas — Edit Mode` (top-left)

**Voice-over cue**: "Your AI evaluation pipeline. Visual. Customizable. No code."

---

### [0:03–0:06] Load a Template

**Screen**: CanvasToolbar → Gallery button

**Action**:
1. Click "Gallery" icon in CanvasToolbar (or template selector)
2. Template picker appears — grid of 5 templates
3. Click "Quick Evaluate" template

**Annotation**: Arrow to Gallery: `5 pipeline templates →`

**Voice-over cue**: "Start from a template."

---

### [0:06–0:10] Nodes Auto-Arrange

**Screen**: Canvas populates with pipeline nodes in sequence

**Action**: Nodes animate into position:
- **Scout** (left) → **Router** → **Draft** → **Critic** → **Queen** → **Archivist** (right)
- Connection edges draw themselves between nodes
- Brief settle animation

**Annotation**: Each node gets a subtle label pulse as it appears:
`Scout → Router → Draft → Critic → Queen → Archivist`

**Voice-over cue**: "Six agents. Pre-wired. Ready to customize."

---

### [0:10–0:14] Interact with a Node

**Screen**: Critic node selected

**Action**:
1. (0:10) Click and drag Critic node slightly to reposition (shows it's movable)
2. (0:12) Click to select — parameter panel slides in from right
3. (0:13) Panel shows: tradition selector, weight sliders (L1-L5), mode toggle

**Annotation**:
- On drag: `Drag to rearrange`
- On panel: `Configure each agent's parameters`

**Voice-over cue**: "Drag to rearrange. Click to configure. Every agent is customizable."

---

### [0:14–0:18] Add a New Node

**Screen**: CanvasToolbar → "Add Node" action

**Action**:
1. (0:14) Click "Add Node" button (+ icon) in toolbar
2. (0:15) Node type picker appears — select "Report"
3. (0:16) New Report node appears on canvas (slight bounce animation)
4. (0:17–0:18) Drag from Report node's output port → connect to Archivist's input port
   - Connection edge draws with satisfying snap

**Annotation**:
- `+ Add any node type`
- On connection: `Connect with drag & drop`

**Voice-over cue**: "Add new nodes. Wire them up. Build exactly the pipeline you need."

---

### [0:18–0:21] Switch Cultural Variant

**Screen**: Pipeline-level settings or tradition selector

**Action**:
1. (0:18) Click pipeline settings or tradition dropdown
2. (0:19) Select "chinese_xieyi" variant from dropdown
3. (0:20–0:21) Visual feedback — nodes briefly flash/update to reflect cultural variant change
   - Color accent shifts subtly

**Annotation**: `Switch cultural tradition → Pipeline adapts`

**Voice-over cue**: "Switch cultural traditions. The entire pipeline adapts."

---

### [0:21–0:24] Execute the Pipeline

**Screen**: CanvasToolbar → Run button (▶)

**Action**:
1. (0:21) Click ▶ Run button in CanvasToolbar
2. (0:22–0:24) Execution animation:
   - Each node highlights sequentially: blue pulsing → green completed
   - Connection edges animate data flow (particles or pulse)
   - Progress indicator appears

**Annotation**: `▶ Run — watch your pipeline execute in real-time`

**Voice-over cue**: "Hit Run. Watch every agent execute in sequence."

---

### [0:24–0:28] Execution Progress

**Screen**: Nodes completing one by one

**Action**:
- Scout → green ✓ (0:24)
- Router → green ✓ (0:25)
- Draft → green ✓ (0:26)
- Critic → green ✓ (0:26.5)
- Queen → green ✓ (0:27)
- Report → green ✓ (0:27.5)
- Archivist → green ✓ (0:28)

**Annotation**: Counter in corner: `Agent 1/7... 2/7... 7/7 ✓`

---

### [0:28–0:30] Switch to Results

**Screen**: Transition from Edit to Run mode

**Action**:
1. Click mode switcher → "Run" tab
2. Results view appears — generated candidates + radar chart + scores

**Annotation**:
- Center: `Drag. Drop. Evaluate.`
- Bottom: `vulcaart.art` + `Open Source`

**Voice-over cue**: "Drag. Drop. Evaluate. Your custom AI pipeline — in 30 seconds."
