# VULCA Demo Recording Scripts

Five recording scripts for the VULCA open-source launch campaign. Each script is self-contained with shot-by-shot timing, annotations, voice-over cues, and a companion social media teaser.

## Scripts Overview

| # | Script | Duration | Mode | Target Audience |
|---|--------|----------|------|-----------------|
| 01 | [NoCode — One Sentence, Six Agents](01-nocode-one-sentence.md) | 30s | Run | Non-technical, PMs, researchers |
| 02 | [LowCode — Drag & Drop Pipeline](02-lowcode-drag-pipeline.md) | 30s | Edit → Run | Developers, engineers |
| 03 | [Compare — Cross-Cultural Evaluation](03-compare-cultural.md) | 40s | Compare | Art historians, ethics teams |
| 04 | [Self-Evolution — Terminal Demo](04-self-evolution.md) | 45s | Terminal + Browser | ML engineers, researchers |
| 05 | [Explore & Build — Cultural Knowledge](05-explore-build.md) | 30s | Explore → Build | Cultural researchers, educators |

## Recording Checklist (All Scripts)

- [ ] Chrome 1920×1080, 100% zoom, bookmarks bar hidden
- [ ] Screen recorder: OBS or similar, 1080p, 30fps+
- [ ] Microphone off during recording (voice-over added in post)
- [ ] Demo images downloaded (`cd ../demo-assets && bash download.sh`)
- [ ] Production site accessible: https://vulcaart.art
- [ ] API accessible: https://vulca-api.onrender.com

## Companion Assets

Each script includes a companion social media teaser (EN/ZH tweet + recommended image from `exports/`).
