# Review Context Requests

Vault status: request intake.

Other sessions use this folder conceptually when they need the vault changed.
They may either create a request file here on a request branch, or paste a
filled request packet into the conversation for a context-curator session.

Do not edit protected vault files directly from a normal session.

## Request Flow

1. Read `../LOCK.md`.
2. Copy `TEMPLATE.md`.
3. Fill all fields.
4. Cite evidence.
5. Submit to a context-curator session.
6. Wait for curator acceptance, revision request, or rejection.
