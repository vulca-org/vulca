# Brand-Safe AI Creative Production Visual Brief

Status: draft-public-blocked

## Visual Mood

Editorial product strategy deck with a calm production-room feel: one large workflow surface per slide, sparse claims, neutral brand-safe placeholder assets, and enough operational detail to feel credible without becoming a dense SaaS dashboard.

## Layout Families

| Slide | Layout Family | Dominant Object | Typography Notes |
| --- | --- | --- | --- |
| 1 | comparative model | A wide flow that moves from one hero asset to a grid of campaign variants. | Use a short claim headline, small labels, and one restrained supporting line. |
| 2 | source-boundary ledger | A stacked set of approved inputs: brief, archive references, templates, product shots, copy, and rules. | Label each input plainly; avoid decorative brand callouts. |
| 3 | workflow canvas | A campaign workflow map with nodes for brief, assets, channel rules, generation, review, and packet assembly. | Use concise node labels and a clear left-to-right reading path. |
| 4 | governance board | A control board with columns for brand rules, asset limits, review states, integrations, and handoff. | Keep headings short and align labels to make controls scanable. |
| 5 | variant matrix | A grid by channel, region, language, format, and review status with neutral placeholder thumbnails. | Use compact table text and one visible status style per row. |
| 6 | release gate | A decision board showing validation checks, flagged items, rerun status, and final handoff. | Make the final decision state visually prominent without using alarming colors. |

## Asset Policy

- Allowed assets: abstract workflow shapes, neutral campaign cards, invented product silhouettes, generic channel icons, localization chips, status markers, and generated placeholder thumbnails for fictional brands.
- Allowed source usage: paraphrased source facts, source ids, and generic diagrams inspired by documented workflow concepts.
- Forbidden assets: Coca-Cola visuals, Coca-Cola archive elements, Adobe logos, Adobe screenshots, Adobe UI layouts, source page compositions, campaign imagery from the cited pages, and any recognizable brand trade dress.
- Use a fictional neutral campaign system so the deck demonstrates workflow structure without imitating a real brand.

## Anti-Patterns

- A generic AI image-generation pitch with no workflow, approval, or handoff object.
- A copied Adobe interface or a Coca-Cola-inspired brand treatment.
- A dashboard crowded with small widgets instead of one dominant proof object.
- Unsupported metrics, speed claims, cost claims, or customer outcome badges.
- A magical fully automated pipeline that hides validation, review, and rerun states.
- Internal implementation vocabulary appearing in slide titles, body copy, proof captions, or presenter-facing generation text.
