# Vulca PPT Case Pack v1

This repository-local pack turns public commercial presentation references into reusable Vulca PPT design knowledge. It is not a training dataset, source corpus, scraped article archive, or slide-asset collection.

Copyright boundary: this pack stores source links, short original summaries, and original analysis only. It must not include copied proprietary slide images, full article text, video transcripts, client assets, event artwork, brand marks, or screenshots from the references.

The first deliverable is a Vulca Product Launch Deck generated mostly through editable presentation code. SVGs, native shapes, and generated bitmap assets are supporting material; they are not a substitute for editable slide structure.

Primary workflow:

```text
sources -> original analysis -> rules and tokens -> slide patterns -> deck outline -> baseline and Vulca generation -> review -> comparison report
```
