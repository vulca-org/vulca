# Results

Generated deck artifacts and comparison records are documented here after the case pack is used.

Scratch files stay under `outputs/$THREAD_ID/presentations/`. Only final, reviewed artifacts should be copied into this folder.

Do not store copied proprietary reference assets, source screenshots, full articles, transcripts, or unreviewed generated drafts here.

Use `asset_provenance.json` as the required record for generated bitmap and SVG prompt provenance, source references, license status, and review status before any generated asset is promoted into a final deck artifact.
