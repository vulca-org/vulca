## Figma Config 2025 Identity

Source ID: `figma_config_2025_identity`

Role: visual system reference.

Original summary: Figma's Brand Studio describes Config 2025 as a system that stretched across stage screens, web, signage, badges, physical activations, motion, and product-launch storytelling. The analysis value is the explicit connection between primitives, component-like glyph systems, type hierarchy, color variables, and motion rules.

Why it matters: Vulca needs a design system that can be expressed as rules and code, not just a mood. This reference reinforces the need for reusable primitives, controlled variation, and a system that works across multiple surfaces.

Use boundary: reference analysis only. Do not copy event artwork, photography, glyphs, motion treatments, article text, or proprietary identity assets.
