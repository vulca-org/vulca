Review gate

Pass criteria

SilhouetteThe object must read as one central garden/specimen in under one second.
Material splitMineral base, glass roots, cyan routes, and amber blossoms must be visually distinct.
State potentialThe object needs clear anchor points for scan, bloom, and constellation overlays.
Model budgetThe baseline GLB should stay lightweight and replaceable by generated candidates.
