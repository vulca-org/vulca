\subsection{中期展望（2027 – 2028）}

\begin{itemize}
  \item 成为 \keyword{Claude Code / MCP 生态中图像域的默认 SDK}，类比代码域的 \code{git} 之于 GitHub；
  \item 覆盖 \keyword{20+ 种文化传统}，其中至少 5 种来自社区贡献，形成\keyword{可持续扩展的数据资产}；
  \item \keyword{10+ 家行业客户}完成本地化部署，其中至少 2 家为\keyword{国家级文博 / 出版集团}；
  \item 完成 \keyword{Pre-A 或 A 轮融资}，团队扩展至 25–35 人（含算法 / BD / 实施交付三条线，保持工程核心），同时保持工程质量（tests/LOC、ship 频率）不下滑；
  \item 成为 ACM MM / CVPR / SIGGRAPH 等会议的\keyword{稳定贡献者}，用学术发表反哺产品差异化。
\end{itemize}

\subsection{转化漏斗：从流量到付费的中间指标}

从目前的流量基线（2026-04-20 README refresh 节点：累计 82 views / 31 unique / 6 stars）到
2026 Q4 保守目标（1500+ stars / 月下载 5000+ / 30 付费账户）之间，
\keyword{需要可被周期性核对的中间漏斗指标}，避免"只讲两端不讲过程"。

\begin{center}
\small
\begin{tabularx}{\linewidth}{@{}>{\tabRR}p{3.4cm}>{\tabRR}X>{\tabRR}p{2.4cm}@{}}
\toprule
\rowcolor{tableHead}
\heiti 漏斗阶段 & \heiti 指标 & \heiti 目标（Q4 2026） \\
\midrule
\keyword{曝光} & GitHub views / dev.to 阅读 / Xiaohongshu 曝光 & 月均 20k+ \\
\keyword{知晓} & GitHub stars & 1500+ \\
\keyword{尝试} & PyPI 下载 / \code{claude plugin install} 安装量 & 月均 5000+ \\
\keyword{激活} & 完整运行 \code{/decompose} 等 skill 一次的独立会话数 & 月均 800+ \\
\keyword{留存} & 7 日内二次会话数 / 贡献 issue 或 YAML 的账户数 & 月均 150+ \\
\keyword{付费意向} & Cloud alpha 申请 / 行业客户 POC 接触 & 累计 80+ \\
\keyword{付费转化} & Cloud 正式付费账户 & 30+ \\
\bottomrule
\end{tabularx}
\end{center}

我们\keyword{不假设每一级的转化率}——转化率数据将在 Cloud alpha 阶段采集后进入 Pre-A 材料，
届时形成完整的单位经济模型。天使轮阶段只做"基线 + 目标 + 中间锚点"。
