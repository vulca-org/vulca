\paragraph{算力成本单独说明（精确口径）}
项目已获得 \keyword{Google Gemini + Anthropic} 两套研究级 / 创业级 API credits（非现金、有额度与期限）。
\keyword{覆盖范围需精确区分}——避免 DD 阶段被质疑：

\begin{itemize}
  \item \keyword{Credits 已覆盖}：产品演示、能力验证、内部工程迭代、有限的早期 alpha 客户使用、学术 benchmark 和演示素材生成。预计支撑未来 12-18 个月的\keyword{工程与演示}场景。
  \item \keyword{Credits 未覆盖（进入 COGS）}：Cloud 正式付费客户的大规模模型调用、On-Prem 实施驻场的算力、GPU 运维、credits 到期后的续期议价风险。
  \item \keyword{对冲方式}：multi-provider 抽象层允许一键切到\keyword{本地 ComfyUI 路径}（边际成本接近存储和带宽）。每 1000 MAU 的推理成本敏感性：Cloud 路径 ¥ 3000–5000（credits 覆盖阶段近零）/ 本地路径 ¥ 800–1500（含折旧与运维）——当前为示意区间，Pre-A 阶段以实测数据替换。
\end{itemize}

净效应：\keyword{天使轮资金不必用于 token 成本}，可全部投入人和市场；
但我们\textbf{不}把 credits 误写为"永久算力 de-risk"，这部分真实压力由产品架构（本地优先）和 pricing（按量计费传导）吸收。

\paragraph{为什么不做 SaaS 单位经济假财务}
\begin{itemize}
  \item 在 agent-native 这个新赛道上，\keyword{LTV / CAC / churn 等基础参数还没有可信经验值}，
  任何精确数字都是假精度；
  \item 天使轮的核心判断是"人 + 赛道"，不是"三年后 IRR"；
  \item 我们更愿意给投资人一份\keyword{可验证的里程碑清单}，而不是一张无法质疑的 Excel；
  \item Pre-A 轮再提供经过 12 个月真实数据校准的单位经济模型，届时数据会更有说服力。
\end{itemize}

\subsection{资金使用里程碑承诺}

作为 C 类受众（孵化器 / 政府立项）关心的\keyword{产出承诺}，
我们把天使轮资金与 7 个可验证节点强绑定。\keyword{以下里程碑以本轮交割完成（预计 2026-Q2）为 T+0 起算}，
与第 \ref{sec:roadmap} 章的季度路线图对齐（T+12 月 $\approx$ 2027-Q1 v1.0 发布节点）：

\begin{enumerate}
  \item T+3 月：2 个新 skill shipped（\code{/evaluate} / \code{/inpaint}）；
  \item T+6 月：团队扩张至 6 人 + 1 个灯塔客户 POC 启动 + EmoArt 结果发布；
  \item T+9 月：Cloud alpha 上线、首批付费意向；
  \item T+12 月：\code{v1.0} 发布、至少 1 个行业客户签约；
  \item T+15 月：Pre-A 数据室搭建完成；
  \item T+18 月：Pre-A 启动或现金流持平（只要二者之一达成即视为天使轮成功）；
  \item \keyword{任何一个节点 miss 30 天以上}，主动向投资人汇报与提出修正方案。
\end{enumerate}

\subsection{退出路径示意}

\begin{itemize}
  \item \keyword{主线路径}：成为 agent-native 图像领域头部 SDK + 托管服务，被 AI 基础设施厂商（Anthropic / Google / 国内头部 agent / 云厂商）战略收购；
  \item \keyword{次线路径}：在文博 / 出版 / 设计行业形成垂直壁垒，独立 IPO 或被行业集团收购；
  \item \keyword{学术溢出}：开源项目本身持续产出论文和学术引用，形成\keyword{品牌护城河}；
  \item \keyword{谨慎}：我们不把"下一轮融资"当成退出；它只是中间步骤。
\end{itemize}
