Findings

  • Object-level segmentation works well: background, face_skin, earring, clothing, headscarf all produce meaningful masks
  • Face-part segmentation fails: "eyes", "lips", "nose" all return the entire face — EVF-SAM cannot distinguish sub-face parts on painterly textures
  • Practical layer ceiling: 5-6 layers (background, face/skin, headscarf, earring, clothing + maybe 1-2 more object-level splits)
  • 12-20 layer face-part decomposition is NOT feasible with current EVF-SAM (referring segmentation treats face parts as "the person")
  • Headscarf mask quality: mostly correct but has horizontal stripe noise where face bleeds through — needs morphological cleanup or better prompt

Implications for Phase 1

  • Target 5-6 layers, not 12-20
  • semantic_path is unnecessary at this granularity — keep as optional metadata only
  • VLM auto-planning should output object-level splits, not face-part splits
  • Prompt engineering may improve headscarf-type edge cases (more specific spatial language)
