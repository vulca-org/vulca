# Anatomy: Spline Contemporary 3D Web References

Primitive: layered spatial UI sculpture made from floating interface panels, state tracks, event hotspots, and spatial nodes.

Technique: translate Spline-like 3D web design into code-native primitives: layer stack, state model, timeline morph, and interaction hotspots.

Technique stack:

- Layer stack: five floating panels establish a 3D interface volume.
- Spatial nodes: small anchors show where objects, states, or constraints connect across the scene.
- State tracks: curved paths show timeline or state transitions between panels.
- Event hotspots: warm circular controls mark hover/click event points.
- Morph motion: subtle panel and node movement teaches stateful interactive UI rather than static product illustration.

Construction sequence:

1. Draw depth lines first so the scene reads as a spatial interface.
2. Add five panels with alternating material colors and perspective-like polygon shapes.
3. Place nodes on panel corners and interaction-critical regions.
4. Add state tracks that connect panels and imply timeline/state flow.
5. Add hotspots last so the interaction layer remains visible.
6. Export still, motion, and standalone HTML before promotion.
